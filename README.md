* <code>HTML5</code>

### Key Features
* <code>Register User</code>
* <code>Login User</code>
* <code>See all movies</code>
* <code>Add/Remove movie to/from favorite list</code>
* <code>View userprofile</code>
* <code>Delete Account</code>
